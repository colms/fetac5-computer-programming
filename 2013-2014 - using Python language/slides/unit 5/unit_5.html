Regular expression

  • there are several regular expression shortcut characters
  • \w equals [a-zA-Z0-9_], all alphanumeric characters
  • \W equals [^a-zA-Z0-9_], all non-alphanumeric characters
  • \d equals [0-9], all numeric characters
  • \D equals [^0-9], all non-numeric characters
  • \s equals [\t\r\n\f], all whitespace characters
  • \S equals [^\t\r\n\f], all non-whitespace characters

Class

A class is a grouping of operations and data that the operations use. A class defines each operation, known as a method and acts as a blue print from which objects can be created.

Object

An object in object-oriented programming (OOP) is an instance of a class. An object is the finished product of following the blue print. The object has its own copy of the data defined in the class and can perform all operations specified from the class.

More refactoring

  • There's a lot of duplication between Monster and Hero. They're always doing the same things and it's a pain to update both each time. Let's put what they have in common into one class. We can use further composition but instead, we'll use inheritance.

Inheritance

In object-oriented programming, inheritance is the concept that when a class of objects is defined, any subclass that is defined can inherit the definitions of one or more general classes. Margaret Rouse
  • composition used for "has-a" relationship, e.g. Monster "has-a" position
  • inheritance used for "is-a" relationship, e.g. Monster "is-a" combat character
  • inheritance is a very powerful tool
  • prefer composition to inheritance

Inheritance

from position import *
        
class CombatCharacter(object):
    _pos = None
    _name = None
    _health = None
    _speed = None
    _defense = None
    _strength = None
    
    def __init__(self, name, strength):
        self._pos = Position(0, 0)
        self._health = 100
        self._name = name
        self._speed = 5
        self._strength = strength
        
    def __repr__(self):
        return "name: " + self._name + ", health: " + str(self._health) + ", position: " + str(self._pos)
     
    def move(self, direction):
        self._pos.move(direction, self._speed)

    def take_damage(self, attack_strength):
        self._health -= attack_strength - self._defense

    def attack(self, target):
        self._battle_cry()
        target.take_damage(self._strength)

import sys
from PyQt5.QtCore import *
from PyQt5.QtWidgets import *
from PyQt5.QtGui import *

class ConverterWidget(QWidget):
  _inchesLabel = None
  _inchesLine = None
  _inchesToCmButton = None
  _cmLabel = None
  _cmResultLabel = None
  
  def __init__(self, parent = None):
    super().__init__(parent)
    self.setup_widgets()
    self.setup_layout()
    self.setWindowTitle("Converter")
    
  def setup_widgets(self):
    self._inchesLabel = QLabel("inches:")
    self._inchesLine = QLineEdit()
    self._inchesToCmButton = QPushButton("Convert")
    self._cmLabel = QLabel("centimeters:")
    self._cmResultLabel = QLabel("")
    self._inchesToCmButton.clicked.connect(self.inches_to_cms)

Converter GUI

  def setup_layout(self):
    main_layout = QGridLayout()
    # addWidget(widget, row, column, rowspan = 1, colspan = 1)
    main_layout.addWidget(self._inchesLabel, 0, 1)
    main_layout.addWidget(self._inchesLine, 0, 2)
    main_layout.addWidget(self._inchesToCmButton, 0, 3)
    main_layout.addWidget(self._cmLabel, 1, 1)
    main_layout.addWidget(self._cmResultLabel, 1, 2)
    self.setLayout(main_layout)
   
  def inches_to_cms(self):
    inches = float(self._inchesLine.text())
    cm = inches * 2.54
    self._cmResultLabel.setText(str(cm))

def main():
  app = QApplication(sys.argv)
  window = ConverterWidget()
  window.show()
  exit_code = app.exec_()
  sys.exit(exit_code)

if __name__ == '__main__':
    main()
