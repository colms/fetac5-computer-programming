What is a list?

In Python, a list is a data structure containing an ordered sequence of values where each value is associated with an index denotating the values position in the sequence.

Lists

x = [3, "apples", "cats", 4.91, "apples"]
print(x) # [3, 'apples', 'cats', 4.91, 'apples']
  • an ordered collection of objects of any type
  • can be of any length, including empty
  • can contain duplicates
  • contained within square brackets
  • elements are separated by commas
  • useful for dealing with a huge number of variables
  • no need to create hundreds of variables

  • note that list start at number zero, not one

List enumeration

animals = ["ape", "bison", "crocodile", "dog", "elephant"]
for animal in animals:
    animal = "zebra"
print(animals)
#prints: ['ape', 'bison', 'crocodile', 'dog', 'elephant']
  • all list elements still have their original value
  • remember that animal is only a temporary variable
  • we do not alter the list value, just the temporary variable value

List enumeration

animals = ["ape", "bison", "crocodile", "dog", "elephant"]
for index, animal in enumerate(animals):
    animals[index] = "zebra"
print(animals)
#prints: ['zebra', 'zebra', 'zebra', 'zebra', 'zebra']
  • all list element values are "zebra"
  • animal is still a temporary variable holding copy of value
  • index holds the index number of the list value
  • index starts at 0 and increases by 1 until it reaches 4 because animals has 5 elements

List exercise 5

Populate a list with 5 integers taken from the user using a while loop. After all inputs have been taken, run through the list to check if an element value is less than 10 using a for loop. If a value is less than 10, set it to 0. Finally, display the list.

List exercise 5 solution

numbers = []
sum = 0
num_inputs = 0

while num_inputs < 5:
    number = int(input("Enter an integer: "))
    numbers.append(number)
    num_inputs += 1

for index, number in enumerate(numbers):
    if number < 10:
        numbers[index] = 0
print(numbers)

Simple section navigation

section = None
while section != "go to sleep":
    if section == "at home":
        print("You are in your home")
        section = input("Where would you like to be? ")
    elif section == "at work":
        print("You must use the right key to unlock your office door")
        key = input("Use blue or red key? ")
        if key == "blue":
            print("You opened the office door")
            print("You remember it's a holiday and go home")
            section = "at home"
    else:
        print("You can either be 'at home' or 'at work'.")
        section = input("Where would you like to be? ")
  • this is a very primitive way to direct the user to different parts of the program

Random number generation

import random
rand_float_1 = random.random() # float from 0 to 1
rand_int = random.randint(0, 10) # integer from 0 to 10
rand_float_2 = random.uniform(0, 10) # float from 0 to 10
rand_even_int = random.randrange(0, 11, 2) # number from 0 to 10 with steps of 2

animals = ["ape", "bison", "crocodile", "dog", "elephant"]
rand_animal = random.choice(animals)
rand_animals = random.sample(animals, 2) # two animals

print("Float from 0 to 1:", rand_float_1)
print("Integer from 0 to 10:", rand_int)
print("Float from 0 to 10:", rand_float_2)
print("Even integer from 0 to 10:", rand_even_int)
print("Random animal:", rand_animal)
print("Two random animals:", rand_animals)
print("Animals before shuffle:", animals)
random.shuffle(animals)
print("Animals after shuffle:", animals)

Random number generation exercise 1

Try to guess the secret integer. The secret integer should be selected randomly at the beginning of the program and should have a value from 1 to 5. Have the user guess the secret number as many times as he likes. Inform the user if their guess is too high, low or if it's correct. Exit when the guess is correct.

Random number generation exercise 1 solution

import random

guess = -1
secret_number = random.randint(1, 10)
print("Guess the secret number from 1 to 10!")
while guess != secret_number:
    guess = int(input("Please enter your guess: "))
    if guess > secret_number:
        print("Too high. Try again!")
    elif guess < secret_number:
        print("Too low. Try again!")
print("Correct!")

Random number generation exercise 2

Modify the previous secret number program so that the user only has two incorrect guesses before losing the game. Inform the user of their success or failure.

Random number generation exercise 1 solution

import random

guess = -1
secret_number = random.randint(1, 10)
lives = 2
print("Guess the secret number from 1 to 10!")
while guess != secret_number and lives != 0:
    print("Number of guesses left:", lives)
    guess = int(input("Please enter your guess: "))
    if guess > secret_number:
        print("Too high. Try again!")
        lives -= 1
    elif guess < secret_number:
        print("Too low. Try again!")
        lives -= 1
if lives == 0:
    print("Game over. You lost.")
else:
    print("Game over. You won!")
