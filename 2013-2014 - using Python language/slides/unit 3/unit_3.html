Character set

  • an encoding that maps byte long sequences of bits to symbols, including letters, numbers, punctuation and more.
  • life before a standard set

US ASCII

  • American Standard Code for Information Interchange
  • A standard character set for representing characters from the English alphabet
  • 128 characters
  • 1 byte – 7 bits for character – 1 for parity

US ASCII

US ASCII

  • ordinal set
  • a set with a kind of order structure applied to it
  • the set of months in a year
  • the set of values in the ASCII table is an ordinal set because each is associated with a character
  • 32 unprintable characters followed by 96 printable characters

US ASCII

for index in range(0, 128, 1):
    print(chr(index))

Control characters

  • Control characters are characters reserved for special behaviour
  • Python has mapped certain control characters to elements in the US-ASCII table
  • put a control character inside a printed string to activate it
    • charactereffect
    \nline feed
    \ttab
    \abeep

Control characters

print("line\nfeed.") # new line
print("Black and \twhite") # tab
print("\a") # beep

Extended ASCII

  • ASCII later unofficially extended to non-English characters as a superset
  • used for glyphs such as áéíóú£
  • used the last parity bit to create 128 more characters
  • many different standards
  • ISO Latin-1
  • Not used so much in Python
  • Unicode (UTF-8) is the default for Python 3

What is a string?

A sequence of data values, usually bytes, which usually stand for characters (a ‘character string’). The mapping between values and characters is determined by the character set which is itself specified implicitly or explicitly by the environment in which the string is being interpreted. Accessing list elements - Advanced

animals = ["ape", "bison", "crocodile", "dog", "elephant"]
print(animals[0:2]) # ['ape', 'bison']
print(animals[1:4]) # ['bison', 'crocodile', 'dog']
print(animals[2:2]) # []
  • a subset or 'slice' of a list can be accessed using the slicing notation
  • slice = my_list[start_index:end_index]
  • slice includes the elements from the start index up to but not including the end index

Accessing list elements - Advanced

animals = ["ape", "bison", "crocodile", "dog", "elephant"]
print(animals[0:-1]) # ['ape', 'bison', 'crocodile', 'dog']
print(animals[1:-2]) # ['bison', 'crocodile']
  • a negative index specifies the distance from the last element in a list

Accessing list elements - Advanced

animals = ["ape", "bison", "crocodile", "dog", "elephant"]
print(animals[2:]) # ['crocodile', 'dog', 'elephant']
print(animals[:2]) # ['ape', 'bison']
print(animals[:-1]) # ['ape', 'bison', 'crocodile', 'dog']
print(animals[-1:]) # ['elephant']
print(animals[:]) # ['ape', 'bison', 'crocodile', 'dog', 'elephant']
  • an unspecified index means every element

String exercise 2

Write a program to count the how many times the letter ‘s’ occurs in the word “Mississippi”. Have the program count the letters using two different techniques: one that you have implemented yourself and another using standard library functions.

String exercise 2 solution

word = "Mississippi"
s_counter = 0
for letter in word:
    if letter == 's':
        s_counter += 1
print("The number of times the letter s occurs in Mississippi is", s_counter)

# The standard library way
print(word.count('s'))

String exercise 4

Write a program to replace all occurrences of the 'i' with 'a' in the word “Mississippi”. Do this using two different techniques: one that you have implemented yourself and another using standard library functions.

String exercise 4 solution

word = "Mississippi"
new_word = ""
for letter in word:
    if letter == 'i':
        new_word += 'a'
    else:
        new_word += letter
print("new_word is", new_word)

# The standard library way
print(word.replace('i', 'a'))

String exercise 5

Generate a random string of 8 characters and output whether the string contains the letter 'e'.

String exercise 5 solution

import random
random_word = ""
for i in range(0, 8):
    rand_int = random.randint(97, 123)
    random_word += chr(rand_int)
print(random_word)
if 'e' in random_word:
    print("e is in", random_word)
else:
    print("e is not in", random_word)

File Input / Output

  • read from and write to files
  • use the "open" function
  • first argument is file name
  • second argument is a letter indicating what we are opening the file for
  • read from existing files - "r"
  • create new files - "w"
  • append to existing files - "a"
  • both read from and wrote to file - "r+"

Reading from a file

  • if the file being read is in the same directory as the python for being executed, we only need to specify the name of the for and not the location
  • paste the animals.txt file into the same directory as the executed python file for the duration of these

Reading from a file

# create handle to input file
# r indicates that we are reading
input_file = open("animals.txt", "r")

# run through the file and print it line by line
for line in input_file:
    print(line)

input_file.close()
input_file = open("animals.txt", "r")

# read the entire file at once as a string
file_string = input_file.read()

# and remove the newline
# at the end of each line in the file
lines = file_string.splitlines()
print(lines)

input_file.close()

Writing to a file

  • if no file exists, one will be created
  • if a file exists, it will be completely overwritten

Writing to a file

# create handle to output file
# w indicates that we are writing
output_file = open("test.txt", "w")
output_file.write("hello\n") # method 1
print("other method",  file=output_file) # method 2
output_file.close()

Appending to a file

# create handle to output file
# a indicates that we are appending
output_file = open("test.txt", "a")
output_file.write("another\n") # method 1
print("few lines", file=output_file) # method 2
output_file.close()

File I/O exercise 1

Write that spell checker program that tells the user if the animal name they have entered is spelled correctly. The spell checker simply searches for a word in a text file and informs the user of a misspelling if the word the user emerged is not in the text file.

File I/O exercise 1 solution

input_file = open("animals.txt", "r")
file_string = input_file.read()
lines = file_string.splitlines()
user_input = input("Enter the animal name you would like to spell check: ")
if user_input in lines:
    print(user_input, "is spelled correctly")
else:
    print(user_input, "is spelled incorrectly")
